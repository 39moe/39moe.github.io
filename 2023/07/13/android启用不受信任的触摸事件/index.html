Android启用不受信任的触摸事件

发表于2023-07-13|更新于2024-10-13|Androidandroid系统相关
|浏览量:

启用安卓不受信任的触摸事件:
解决了如果有Toast存在无法打字的问题或者其他东西存在无法触摸的问题:

root在终端输入: settings put global block_untrusted_touches 0

无root需要借助adb:
adb shell settings put global block_untrusted_touches 0
